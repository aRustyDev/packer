# OpenStack Glance Image

- OpenStack Glance Image:
  - Accepts multiple formats: raw, qcow2, vhd, vhdx, vmdk, vdi, iso, ami/ari/aki (legacy). Glance tracks disk_format and container_format (bare, ovf, docker, etc).
